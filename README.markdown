MockVersion
===========
A mock for the svn command line, created to help tool development

Installation
------------
    gem install mockversion

Usage
-----
    require 'mockversion'

    `svn help`
    => Mockversion - A mock for the svn command line, created to help tool development"

    `svnadmin help`
    => Mockversion - A mock for the svn command line, created to help tool development"
